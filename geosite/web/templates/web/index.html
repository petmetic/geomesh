This app is for professionals who need exact terrain modeling from surveyors' data.

It translates points from .csv file to a .txt file.
For instructions on how to extract points from an AutoCAD file, please see tutorials for AutoCAD.

Please be aware of how the 'z' coordinate is written! You might have to add an attribute to get the height of the coordinates.
After you have acquired a .csv file, please upload the file and press submit.
After your .csv file is processed in a few seconds, please use the download link to get your .txt file.
